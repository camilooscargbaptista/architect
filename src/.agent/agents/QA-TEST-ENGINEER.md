## Testing Process

1. **BDD first** — Gherkin scenarios before code
2. **TDD** — RED → GREEN → REFACTOR
3. **Coverage** — verify after each implementation
4. **Regression** — ALL previous tests must pass
5. **Review** — tests are reviewed alongside the code
