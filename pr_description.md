- 🌐 **HTML Interactive Output**: The `HtmlReportGenerator` now dynamically processes fallback abstractions within the TUI itself. This prevents `.map` crashes from empty matrices and successfully compiles the final visual Dashboard to the user (`architect-report-genesis.html`).
- 🔗 **VS Code Integration**: Updated the `architect-vscode` extension to trigger the new Genesis command (`npx -y @girardelli/architect@latest genesis`) natively mapped to the workspace root directory.
- 🧹 **Legacy v3.1 Purge**: Executed a global script to migrate legacy signature strings and prompt bases across the workspace, upgrading all *Generated by Architect v3.1* references to *Architect v8.1*.
